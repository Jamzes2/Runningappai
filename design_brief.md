# RunSynergy Web App Design Brief (Updated)
Version: 1.1

Platform: Web App Dashboard
Brand: RunSynergy

---

# 1. DESIGN GOAL

The web app should feel like:
- A professional running app
- Modern sports analytics software
- Elite athlete dashboard
- Clean and intelligent
- Premium and futuristic

The UI should combine:
- Strava
- Whoop
- Apple Fitness
- Linear
- Vercel dashboard aesthetics

---

# 2. BRAND SYSTEM

## Primary Colours

| Usage | Hex |
|---|---|
| Primary Accent (Bright Green) | #C4FF00 |
| Background (Black) | #000000 |
| Secondary Background (Dark Gray) | #202020 |
| Card Background | #111111 |
| Text Primary | #FFFFFF |
| Text Secondary | #A0A0A0 |

## Logo
- Custom slanted "RS" glyph with dynamic horizontal speed lines trailing the left side of the 'R' and the top of 'S'. 
- Primarily rendered in Bright Green (#C4FF00).

## Typography

Font Family:
- Montserrat (Regular / Bold)

---

# 3. BUTTON STYLES
Buttons are styled as pill shapes (`border-radius: 9999px`) with Montserrat font:
- **Primary Pill**: Solid Bright Green (#C4FF00) background, bold black text.
- **Secondary Pill**: Outlined with thin Bright Green (#C4FF00) border, transparent background, bright green text.
- **Accent Dark Pill**: Outlined with thin dark green/gray border, black background, subtle green glowing text.
- **Disabled Pill**: Dark slate gray, low contrast.

---

# 4. ICONOGRAPHY
- Minimalist circular container design.
- Outline style icons (Speedometer, Stopwatch, Lightning, Gear, Tools, Location Pin, Bullseye, Trophy) encapsulated inside matching circular green frames/borders.

---

# 5. OVERALL WEB APP LAYOUT

## Structure

Desktop layout:
- left sidebar navigation (including custom RS logo, profile, subscription badge)
- Top navigation bar (frosted glass, page title, search bar, sync, notifications)
- Main dashboard content (customisable cards with glassmorphism, glowing curves)
- Floating AI assistant/chat panel

---

# 6. SIDEBAR DESIGN

- Thin borders

## Sidebar Navigation

Include:

# 7. MAIN DASHBOARD

## Dashboard Components

Include Cards For:
- Recent run (with map trace & primary metrics)
- Total weekly km’s
- Heart rate zones (interactive zones)
- AI insights (smart coaching bullets)
- Recovery score (radial ring glow meter)
- Cadence analysis
- Pace progression
- Training load
- Running form analysis
